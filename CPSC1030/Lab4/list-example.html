TCP
Transmission Control Protocol is a method (protocol) used along with the Internet Protocol(IP) to send data in the form of message units, called packets, between two computers over the internet.
IP
Internet Protocol is the method or protocol by which data is sent from one computer to another on the Internet. Each computer on the Internet is uniquely identified by an IP address.
